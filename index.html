Common CSS properties

Name Description
Color It allows us to specify the text color of an HTML element.
Font-Size It allows us to change the size of the font for any text based content.
Background color It allows you to set the background color of an HTML element. Just like text color, the value of this property can be a hex value, rgba or named color.
Height/Width These properties specify the height and width of a HTML element respectively.
Border This property allow us to set a border for our HTML document.
Backgroud Image This property allows us to add a background image to an HTML element.
Border-Radius This property allows you to add rounded corners of a HTML elements.
